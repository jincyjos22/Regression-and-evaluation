# Regression-and-evaluation
# California Housing Regression Assignment

## 📌 Objective

The objective of this assignment is to apply different regression algorithms on the California Housing dataset and evaluate their performance using various evaluation metrics.

---

## 📊 Dataset

* California Housing dataset from sklearn
* Contains features like income, house age, population, etc.
* Target variable: Median house value

---

## 🤖 Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor
* Gradient Boosting Regressor
* Support Vector Regressor (SVR)

---

## ⚙️ Techniques Applied

* Data preprocessing
* Feature scaling using StandardScaler
* Train-test split
* Model training and prediction
* Model evaluation using:

  * Mean Squared Error (MSE)
  * Mean Absolute Error (MAE)
  * R² Score
* Cross-validation (k=5)
* Hyperparameter tuning using GridSearchCV

---

## 🏆 Best Model

The best-performing model is **Gradient Boosting (or Random Forest based on results)** because it achieved the highest R² score and lowest error values.
